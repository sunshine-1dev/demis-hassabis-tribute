# Demis Hassabis Tribute

A tribute page for **Demis Hassabis**, the co-founder and CEO of DeepMind, inspired by the DeepMind website aesthetic.

## Features

- DeepMind-inspired visual design with aurora gradients
- Responsive layout optimized for all devices
- Interactive scroll animations
- Glassmorphism cards and modern typography

## Live Demo

Visit the site at: https://sunshine-1dev.github.io/demis-hassabis-tribute

## About Demis Hassabis

- Co-founder and CEO of DeepMind (2010)
- Nobel Prize in Chemistry (2024) for AlphaFold
- Creator of AlphaGo, the first AI to defeat a world champion at Go
- Fellow of the Royal Society (FRS)

## Design

The page captures DeepMind's signature aesthetic:
- Dark theme with flowing purple-blue-cyan gradients
- Aurora blob animations
- Glassmorphism cards with backdrop blur
- Inter font family
- Smooth scroll-triggered reveal animations
